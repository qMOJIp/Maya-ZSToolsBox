# Maya脚本管理器

![Maya脚本管理器](https://i.imgur.com/PLACEHOLDER.jpg)

## 简介

Maya脚本管理器是一个用于Maya的工具，可以帮助您更高效地组织、管理和运行Python和MEL脚本。它提供了直观的界面，让脚本管理变得简单而高效。

### 主要功能

- **脚本管理** - 创建、编辑、组织和运行Python/MEL脚本
- **分组功能** - 将脚本整理到不同分组中，便于管理
- **一键执行** - 点击按钮即可运行脚本
- **拖放支持** - 直接拖放外部.py或.mel文件导入
- **回收站机制** - 删除的脚本可从回收站恢复
- **脚本提示** - 鼠标悬停时显示脚本功能说明
- **配置导入/导出** - 备份和共享您的脚本集合
- **文件夹管理** - 可在tool文件夹下添加文件夹，然后放入对应脚本，然后回maya点刷新

### 兼容性

- 支持Maya 2022-2025
- 兼容PySide2和PySide6
- 支持Windows、MacOS和Linux系统

## 安装指南

### 方法1：通过拖放安装（最简单）

1. 下载完整的脚本管理器包
2. 解压下载的文件到任意位置
3. 打开Maya
4. 将`install_scripts_box.py`文件拖放到Maya的视口中
5. 安装程序会自动完成所有设置

### 方法2：手动安装

1. 下载完整的脚本管理器包
2. 解压下载的文件到您的Maya脚本目录，例如：
   - Windows: `Documents\maya\scripts`
   - MacOS: `~/Library/Preferences/Autodesk/maya/scripts`
   - Linux: `~/maya/scripts`
3. 在Maya中运行以下Python代码：

```python
import scripts_box
scripts_box.show_scripts_box()
```

4. (可选) 创建架子按钮或添加到工具架中：

```python
# 添加到工具架中
import maya.cmds as cmds
cmds.shelfButton(
    command="import scripts_box; scripts_box.show_scripts_box()",
    image="commandButton.png",
    label="脚本管理器",
    annotation="打开Maya脚本管理器"
)
```

## 使用指南

### 基本操作

#### 脚本管理

- **创建脚本**：点击顶部工具栏中的"新建脚本"按钮
- **运行脚本**：点击脚本按钮
- **编辑脚本**：右键点击脚本按钮，选择"编辑脚本"
- **删除脚本**：右键点击脚本按钮，选择"删除"
- **修改提示信息**：右键点击脚本按钮，选择"编辑提示信息"

#### 分组操作

- **创建分组**：点击左侧导航栏底部的"+"按钮
- **重命名分组**：右键点击左侧导航栏中的分组按钮，选择"重命名"
- **删除分组**：右键点击左侧导航栏中的分组按钮，选择"删除"
- **移动脚本**：右键点击脚本按钮，选择"移动到分组"→选择目标分组

#### 导入与导出

- **导入脚本**：直接将.py或.mel文件拖放到窗口中
- **导出配置**：点击顶部"设置"按钮，选择"导出配置"
- **导入配置**：点击顶部"设置"按钮，选择"导入配置"

### 高级功能

#### 文件夹管理

1. 进入脚本管理器安装目录
2. 打开`tool`文件夹
3. 在其中创建或修改子文件夹（每个子文件夹对应一个分组）
4. 将脚本文件(.py或.mel)放入对应的文件夹中
5. 返回Maya，点击"刷新"按钮更新界面

#### 回收站操作

- **查看回收站**：点击左侧导航栏底部的"回收站"按钮
- **恢复脚本**：在回收站中点击脚本卡片上的"恢复"按钮
- **永久删除**：在回收站中点击脚本卡片上的"删除"按钮
- **清空回收站**：在回收站顶部点击"清空回收站"按钮

## 常见问题

1. **无法运行脚本**：检查脚本语法是否正确，Maya版本是否兼容
2. **脚本丢失**：检查回收站，或检查电脑中是否有备份
3. **无法导入配置**：确保导入的配置文件格式正确，且包含必要的工具目录结构
4. **提示"找不到工具文件"**：尝试点击刷新按钮，或检查文件是否存在于正确的文件夹中

## 提示和技巧

- 在脚本中添加注释，可以自动生成提示信息：
  - Python脚本: `# 提示: 这是脚本的功能说明`
  - MEL脚本: `// 提示: 这是脚本的功能说明`

- 使用导出配置功能定期备份您的脚本集合

- 为经常使用的脚本创建单独的分组，提高工作效率
